Deployment Options

NAKIVO Backup & Replication is a versatile solution that can be installed on most modern operating systems and hardware solutions:

  • Windows

  • Linux

  • AWS EC2

  • FreeNAS

  • Synology NAS

  • QNAP NAS

  • Western Digital NAS

  • ASUSTOR NAS

  • NETGEAR ReadyNAS

  • ARM-based Device

  • Raspberry Pi

For the full list of supported systems and devices, refer to the System Requirements section.
