First Steps with NAKIVO Backup & Replication

When you log in to the application for the first time, the initial configuration wizard opens.

Proceed as follows:

  1. On the Inventory page of the wizard, click Add New.

  2. Select one of the options below:

    • VMware vCenter or ESXi host

    • Microsoft Hyper-V host or cluster

    • Physical machine(s)

    • Microsoft 365 account

    • Nutanix AHV cluster

    • AWS/Wasabi account

    • Oracle database

  1. Proceed with adding items as described in the Inventory article. 

  2. On the Transporters page of the wizard, you will find information about the Transporter component of the NAKIVO Backup & Replication.

  3. To deploy a new Transporter or add an existing one, click Got it and proceed as described in the Transporters article.

  4. To move to the next page of the wizard, click Next.

  5. On the Repositories page of the wizard, you can add a local or a remote Backup Repository to your application by clicking Add Backup Repository.  

  6. Click Finish. 

  7. The Jobs dashboard of the application opens. Proceed with creating your backup and replication jobs.

 

If your current license type is Free and the Trial license has not yet been applied to the current deployment of NAKIVO Backup & Replication, a dialog box appears. Using this dialog box, you can contact the sales team to change your license type or try the full functionality of the solution for 15 days. If you do not want to upgrade your license type right away, you can do it at any time in the Help menu.

Note

If you switch the license type to Trial, the product will automatically go back to using your Free license after expiration.
