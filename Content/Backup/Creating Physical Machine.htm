Creating Physical Machine Backup Jobs

With NAKIVO Backup & Replication, you can back up both Windows and Linux physical machines. This can be done by creating a physical machine backup job that specifies which machines should be backed up, where the backups should be located, how often the backup should occur, and what backup options should be used. To create a physical machine backup job, click Create on the Dashboard and then click Physical machine backup job.

 

The New Backup Job Wizard for Physical machine opens. Complete the wizard as described in the sections below:

  • Backup Job Wizard for Physical Machine: Machines

  • Backup Job Wizard for Physical Machine: Destination

  • Backup Job Wizard for Physical Machine: Schedule

  • Backup Job Wizard for Physical Machine: Retention

  • Backup Job Wizard for Physical Machine: Options
