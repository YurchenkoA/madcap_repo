Nutanix AHV VM Recovery

With NAKIVO Backup & Replication, you can recover full VMs from backups. Multiple VMs can be recovered by a single recovery job. The system allows selecting separate resources (clusters, storage containers, and networks) for different VM.

Important!

When you run VM recovery, a new VM is created—the source VM is not reverted to a previous state or replaced with the new VM.
