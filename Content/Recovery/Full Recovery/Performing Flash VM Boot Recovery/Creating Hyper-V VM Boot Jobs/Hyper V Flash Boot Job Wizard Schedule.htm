Hyper-V Flash Boot Job Wizard: Schedule

On the Schedule page of the wizard, select to run the recovery job manually or schedule the job to run on a regular basis. Proceed as described in the sections below:

  • Disabling Scheduled Job Execution

  • Daily Job Execution

  • Monthly or Yearly Job Execution

  • Periodic Job Execution

  • Chained Job

  • Additional Schedule

Disabling Scheduled Job Execution

If you want to start the job manually only (without any schedule), select the Do not schedule, run on demand checkbox and click Next.

Daily Job Execution

To run a job once a day, do the following:

  1. Choose a time zone to be used for the job start and end times from the time zone drop-down list.

  2. Choose Run daily/weekly from the schedule drop-down list.

  3. Specify the time when the job should be started in the Starting at: box.

  4. Specify the end time for the job in the Ending: box. If the job has not completed by the time specified, the job will be stopped.

  5. Select the days of the week on which the job will be started.

  6. If necessary, select the Effective from checkbox and pick the date when the schedule should come into effect. 

  7. Click Next.

     

Monthly or Yearly Job Execution

To run a job monthly or yearly, do the following:

  1. Choose a time zone to be used for the job start and end times from the time zone drop-down list.

  2. Choose Run monthly/yearly from the schedule drop-down list.

  3. Specify a job start schedule in the appropriate boxes.

  4. Specify the time when the job should be started in the Starting at: box.

  5. Specify the end time for the job in the Ending: field. If the job has not completed by the time specified, the job will be stopped.

  6. If necessary, select the Effective from checkbox and pick the date when the schedule should come into effect. 

  7. Click Next.

Periodic Job Execution

To run a job multiple times per day, do the following:

  1. Choose a time zone to be used for the job start and end times from the time zone drop-down list.

  2. Choose Run periodically from the schedule drop-down list and then choose a time period from the appropriate boxes.

  3. Specify the time when the job should be started in the Starting at: box.

  4. Specify the end time for the job in the Ending box. If the job has not completed by the time specified, the job will be stopped.

  5. Select the days of the week on which the job will be started.

  6. If necessary, select the Effective from checkbox and pick the date when the schedule should come into effect. 

  7. Click Next.

     

Chained Job

To run the job after a previous one has completed, choose Run after another job from the schedule drop-down list and set the options as follows:

  1. After the job: Select a job after which the current job will be started.

  2. Run this job: Choose whether to run the current job immediately after the previous one has completed or specify a delay.

  3. After successful runs: When selected, the job will run if the previous one has completed successfully.

  4. After failed runs: When selected, the job will run if the previous one has failed.

  5. After stopped runs: When selected, the job will run if the previous one has been stopped.

  6. If necessary, select the Effective from checkbox and pick the date when the schedule should come into effect. 

  7. Click Next.

Additional Schedule

If you need to add an additional schedule, click Add another schedule and configure it as described above.
