Creating Flash Boot Jobs for Physical Machines

To create a Flash VM Boot job for physical machines, do one of the following:

  • Create a Flash VM Boot job from the Dashboard by clicking Recover and then selecting Flash VM Boot for physical machines.

  • Create a Flash VM Boot job from the Repositories tab in Settings by following the steps below:

    1. Go to the main menu of NAKIVO Backup & Replication and click Settings.

    2. Go to the Repositories tab and hover over a Backup Repository containing the required backup.

    3. Click Recover and then click Flash VM Boot for physical machines.

The New Flash Boot Job Wizard for Physical Machines opens.
