Flash Boot Job Wizard for Physical Machines: Backups

On the Backups page of the wizard:

  1. In the left pane, select one or more physical machine backups using one of the following views:

    • Jobs & Groups

    • Backup Repositories

  2. In the left pane and then select a recovery point for each backup in the right pane. 

  3. Click Next to go to the next page of the wizard.
