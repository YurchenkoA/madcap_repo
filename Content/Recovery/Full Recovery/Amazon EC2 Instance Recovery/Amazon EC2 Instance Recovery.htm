Amazon EC2 Instance Recovery

With NAKIVO Backup & Replication, you can easily recover Amazon EC2 instances from their backups. When you run the recovery, a new Amazon EC2 instance is created, that is, the original Amazon EC2 instance (if it exists in the recovery location) is not reverted to a previous state or replaced with the new instance.
