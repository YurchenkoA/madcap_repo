Object Recovery for Microsoft Active Directory

The object recovery feature allows you to browse, search and recover Microsoft Active Directory objects directly from compressed and deduplicated backups. This feature is agentless, works right out of the box, and does not require you to create a special lab or run a special type of backup. Microsoft Active Directory objects can be recovered in .ldif format and then be imported back to the Active Directory Server.
